Control de presión arterial

Esta aplicación sirve para llevar un control de la presión arterial.
  • "Conectarse" se utiliza para ingresar, con lo cual se puede dar de alta nuevos valores y observar el gráfico.
  • En "Registrarse" se puede dar de alta un nuevo usuario.

    • Una vez conectado:
  • En el ítem "Datos" permite ingresar los valores de presión mínima y máxima.
  • En el ítem "Graficos" se encuentra un gráfico realizado con los valores ingresados.

Esta aplicación está desarrollada con Angular 7 y Firebase.
